# **App Name**: Prasthan Connect

## Core Features:

- Firebase Authentication: Implement Firebase Authentication for user login and signup with email/password.
- User Header Display: Display user's full name and initials in the header upon successful login, replacing login/signup buttons with a user initials dropdown (Profile & Logout).
- Static Content Pages: Implement static pages for Home, About, Gallery, Team, and Contact, using hardcoded content with responsive design.
- Image Gallery with Filtering: Create a grid-based gallery with filtering options (All, Diwali, Holi, Events, Charity), displaying images and videos in a lightbox popup.
- Contact Form: Implement a contact form with fields for Full Name, Email, Contact Number, and Message, displaying a success alert upon submission.

## Style Guidelines:

- Primary color: Red (#CC0000) for buttons, highlights, and headings, reflecting Prasthan Group's brand identity.
- Dark Gray (#212121) for text, footer, and dividers to provide contrast and readability.
- Background color: White (#FFFFFF) to ensure clean and modern look
- Headings and Titles font: 'Raleway' (sans-serif) for a modern and easily readable style. Note: currently only Google Fonts are supported.
- Paragraphs and Body font: 'Open Sans' (sans-serif) for readability in body text. Note: currently only Google Fonts are supported.
- Use flexbox and CSS Grid for responsive layouts across all sections.
- Max width: 1200px with centered content on small devices. Consistent section padding of 80px.
